# Bloom & Vibe - Fashion Store for Gen Z

A modern, aesthetic fashion website designed for girls aged 15-19.

## Features
- **Modern Aesthetic Design**: Clean layout with pastel accents and trendy typography.
- **Responsive**: Works on mobile and desktop.
- **Interactivity**: Smooth scrolling, hover effects, and fade-in animations.
- **Pages Included**: Home, Shop, About, Contact.

## How to Run
Simply open `index.html` in your web browser (Chrome, Edge, etc.).

## Technologies Used
- HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (Vanilla)
- Google Fonts (Outfit)
- FontAwesome (Icons)
